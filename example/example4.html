Live demo for skin selection

Click, to activate / switch color sets and layout types.
You will see that the player immediately changes.
Additionally have a look to the html preview section below - this also changes the required code.

Color set:

Player Color With the shipped CSS you get a default skin.
You can create custom skins with overriding the CSS elements. To give you an entry point you can have a look to the css files in the skins directory.

 Type:

Layout Types The player is shipped with 2 style types: window like and as bottom bar.
The bottom bar is activated in the same way: by loading a css file. That it is fixed is done with css property bottom:0

remark: If you use a style with drag and drop support (for playlist, download window) but the main window must be unmovable, then you can switch it with true/ false: 

oMcPlayer.makeMainwindowMovable(false);
