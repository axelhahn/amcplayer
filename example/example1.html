Preparation

How to run the player?
In your html document you need to add a css file in the header and a javascript at the end.
After loading the player javascript file you can init the player.

And one little detail: next to the plain player there is a javascript that enables drag and drop functionality. Just load it before the player javascript file.

Remark: for all CSS and JS files exist original sources and minified versions as well. All examples use the minified files.

Html preview

<head>
  (...)
  <link rel="stylesheet" type="text/css" href="../mcplayer.min.css" media="screen" />
</head>
<body>

  (...)

Drag and drop functionality

By default there is no drag and drop functionality of the window like player.
The player class detects the existance of my class addi (which is part of the download).
You just need to load it before mcplayer.js.

In this demo:

  1. Move your mouse pointer to the main player window. Hold the mouse key down and move the player window around.
  2. Open a window i.e. playlist or download - you can move these windows too.
  3. Press [F5] to reload the window. The windows you moved keep their position.

Add audios

stereo audios

For testing you can begin with with a single audio format. I suggest to use mp3.

<audio loop="loop"
	controls="controls"
	preload="none"
	title="[Title]"
	>
	
	<source src="[song-in-stereo].mp3" type="audio/mp3" data-ch="2.0" title="2.0" />

</audio>

Surround Audio formats

The special feature of this player is handling sourrund audios. Each songtitle must exist in stereo and surround in 2 formats for both channel types.

  • stereo:
    • Ogg - Ogg Theorea, stereo
    • Mp3 - MPEG I Layer 3 audio file (the classic mp3 file)
  • 5.1:
    • M4a - MPEG 4 audio (aac)
    • Ogg - Ogg Theorea, 5.1

How to create the different file types?
I recommend to automize the conversion of 2 exports (stereo, surround) of your DAW.
I myself use a few batch files using ffmpeg, oggenc2 and neroAacEnc.

Html preview

For each audio in your website you need a audio tag with several sources for all channels.
Also pay attention to the attributes to group the channels. 

<audio loop="loop"
	controls="controls"
	preload="none"
	title="[Title]"
	>
	
	<!-- stereo sources -->
	<source src="[song-in-stereo].mp3" type="audio/mp3" data-ch="2.0" title="2.0" />
	<source src="[song-in-stereo].mp3" type="audio/mp3" data-ch="2.0" title="2.0" />

	<!-- surround sources -->
	<source src="[song-in-5.1-surround].m4a" type="audio/mp4" data-ch="5.1" title="5.1"  />
	<source src="[song-in-5.1-surround].ogg" type="audio/ogg" data-ch="5.1" title="5.1" />

	<!-- for fallback add a flashplayer here -->

</audio>
